# Управление экраном в NixOS с Hyprland

## Что было настроено

Для отключения автоматического выключения экрана при бездействии были добавлены следующие настройки:

### Системные настройки (modules/system/)

1. **power.nix** - основные настройки управления питанием:
   - Отключение автоматического сна
   - Игнорирование закрытия крышки ноутбука
   - Настройка CPU governor на performance

2. **wayland.nix** - настройки Wayland:
   - Переменные окружения для отключения автоматического выключения экрана
   - Настройки upower для управления питанием

### Пользовательские настройки (modules/hm/)

1. **scripts.nix** - скрипты для управления экраном:
   - `screen-off` - выключить экран
   - `screen-on` - включить экран
   - `toggle-screen` - переключить состояние экрана
   - `disable-screen-blanking` - отключить автоматическое выключение экрана

2. **Горячие клавиши в Hyprland**:
   - `SUPER + F1` - переключить состояние экрана
   - `SUPER + F2` - отключить автоматическое выключение экрана
   - `SUPER + F3` - выключить экран
   - `SUPER + F4` - включить экран

## Применение изменений

После внесения изменений выполните:

```bash
# Пересобрать конфигурацию
sudo nixos-rebuild switch

# Или если используете nh
nh os switch
```

## Использование

### Автоматическое отключение экрана
Экран больше не будет автоматически выключаться при бездействии.

### Ручное управление экраном
Используйте горячие клавиши или команды в терминале:

```bash
# Переключить состояние экрана
~/.local/bin/toggle-screen

# Отключить автоматическое выключение экрана
~/.local/bin/disable-screen-blanking

# Выключить экран
~/.local/bin/screen-off

# Включить экран
~/.local/bin/screen-on
```

### Прямые команды Hyprland
```bash
# Выключить экран
hyprctl dispatch dpms off

# Включить экран
hyprctl dispatch dpms on

# Проверить состояние экрана
hyprctl monitors
```

## Дополнительные настройки

Если нужно временно включить автоматическое выключение экрана:

```bash
# Установить таймер на 5 минут
hyprctl dispatch dpms 300
```

## Устранение проблем

Если экран все еще выключается:

1. Проверьте настройки BIOS/UEFI
2. Убедитесь, что все модули загружены
3. Проверьте логи: `journalctl -f`

## Откат изменений

Если нужно вернуть автоматическое выключение экрана:

1. Удалите или закомментируйте импорты в `modules/system/default.nix`
2. Удалите или закомментируйте импорты в `modules/hm/default.nix`
3. Пересоберите конфигурацию
